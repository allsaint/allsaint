# Hospital Management System

A comprehensive hospital management system built with Flask and PostgreSQL.

## Features

- **Pharmacy Module**: Drug inventory, sales, stock management
- **Billing Module**: Patient billing, payment tracking, revenue reports
- **HR Module**: Staff management, scheduling, attendance tracking
- **Multi-Hospital Support**: Run multiple hospital instances independently
